## 🔍 具体问题分析

### 问题 1：流量极低（最严重）

**现状**：
- 28 天仅 25 次点击，平均每天不到 1 次
- 仅 201 次展示，说明网站在 Google 中曝光极少
- 只有 15 个搜索词，覆盖面太窄

**根本原因**：
1. **内容量太少** - 网站页面数量不足
2. **关键词覆盖不足** - 缺少目标关键词的内容
3. **索引页面少** - Google 收录的页面数量有限

### 1.2 添加结构化数据（Schema.org）

为餐厅页面添加 JSON-LD 结构化数据，提升搜索结果展示效果。

**示例代码**（添加到餐厅详情页 `<head>` 中）：

```html
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "Restaurant",
  "name": "聚龙轩 Julongxuan",
  "image": "https://mxchino.com/assets/julongxuan-cover.jpg",
  "address": {
    "@type": "PostalAddress",
    "streetAddress": "Av. Insurgentes Sur 1971",
    "addressLocality": "Ciudad de México",
    "addressRegion": "CDMX",
    "postalCode": "01020",
    "addressCountry": "MX"
  },
  "servesCuisine": "粤菜",
  "priceRange": "$$$$",
  "telephone": "+52-55-XXXX-XXXX",
  "openingHours": "Mo-Su 11:00-22:00",
  "aggregateRating": {
    "@type": "AggregateRating",
    "ratingValue": "4.5",
    "reviewCount": "120"
  }
}
</script>
```

**需要添加的页面**：
- 所有餐厅详情页（优先：聚龙轩、Mu Lan、远东超市）
- 首页（使用 `WebSite` 类型）

### 2.3 优化现有公司详情页模板

**当前问题**：
- Meta 标题格式单一
- 缺少描述性内容
- 没有用户评价

**优化方案**：

**修改 `company.html` 模板**：

```html
<!-- 动态生成 Meta 标签 -->
<title>{{name}} - {{cuisine}}餐厅 | {{city}} | mxchino.com</title>
<meta name="description" content="{{name}}位于{{city}}{{district}}，提供正宗{{cuisine}}。招牌菜：{{signature_dishes}}。地址：{{address}}。电话：{{phone}}">

<!-- 添加面包屑导航 -->
<nav aria-label="breadcrumb">
  <ol class="breadcrumb">
    <li><a href="/">首页</a></li>
    <li><a href="/{{city}}-restaurants.html">{{city}}中餐厅</a></li>
    <li><a href="/{{cuisine}}-restaurants.html">{{cuisine}}餐厅</a></li>
    <li class="active">{{name}}</li>
  </ol>
</nav>

<!-- 添加 FAQ 部分（提升 SEO） -->
<section class="faq">
  <h2>常见问题</h2>
  <div itemscope itemprop="mainEntity" itemtype="https://schema.org/Question">
    <h3 itemprop="name">{{name}}营业时间是什么？</h3>
    <div itemscope itemprop="acceptedAnswer" itemtype="https://schema.org/Answer">
      <div itemprop="text">{{opening_hours}}</div>
    </div>
  </div>
</section>
```

**检查项**：
- 图片压缩（所有图片 < 500KB）
- 启用 Gzip 压缩
- 使用 CDN
- 延迟加载图片

**工具**：
- Google PageSpeed Insights
- GTmetrix

---

### 4.2 移动端优化

**检查项**：
- 响应式设计
- 触摸友好的按钮（至少 48x48px）
- 避免使用 Flash
- 字体大小至少 16px

**工具**：
- Google Mobile-Friendly Test
