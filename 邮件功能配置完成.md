## 🌐 可用的 API 功能

启动服务器后，以下 API 可以使用：

### 发送邮件
```bash
POST /api/email/send
Content-Type: application/json

{
  "to": "recipient@example.com",
  "subject": "邮件主题",
  "text": "纯文本内容",
  "html": "<h1>HTML内容</h1>"  // 可选
}
```

### 发送测试邮件
```bash

```
